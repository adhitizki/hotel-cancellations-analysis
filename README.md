# Analisis Pembatalan Pemesanan hotel

Repositori ini berisi analisis data Hotel Bookings. Analisis ini dilakukan untuk mengetahui pola pembatalan pemesanan hotel dan faktor-faktor yang mempengaruhinya.


## Tools dan Teknologi
  - Python
  - Pandas
  - Numpy
  - Matplotlib
  - Seaborn
